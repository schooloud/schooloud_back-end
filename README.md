# ☁️ Schooloud (스쿨라우드) ☁️
> 학교 프로젝트 환경 제공용 오픈스택 기반 클라우드 인프라 제공 서비스

클라우드 서비스를 이용해 학교 프로젝트 혹은 동아리 프로젝트를 진행하고 싶지만, 클라우드 비용이 부담되는 학생들을 위한 클라우드 인프라 서비스
